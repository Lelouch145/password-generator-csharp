# Password Generator – C#

A simple console-based password generator built in C#. The program generates a random password based on user preferences such as password length, numbers, and special characters.

This project was created to practice fundamental C# programming concepts such as methods, loops, random number generation, and input validation.

## Features

- Generate random passwords
- Choose password length
- Option to include numbers
- Option to include special characters
- Input validation for user choices

## Example Usage

--- Password Generator ---

How long should the password be: 12  
Include numbers? (y/n): y  
Include special characters? (y/n): n  

Generated password: aB7kLm9QwR2T
